Ampliar

GanttProject es una herramienta de escritorio libre y multiplataforma que permite la realización de diagramas de GANTT a fin de planifica un proyecto y gestionar los recursos. Es una herramienta muy completa a la hora de planificar un proyecto, ya que permite una visualización absoluta del mismo.

Todo queda bajo el control del programa, desde los recursos necesarios en materia personal, así como los días festivos, hasta dividir el proyecto en un árbol de tareas y asignar a cada uno los recursos oportunos.

GanttProject permite establecer dependencias que interrelacionan las tareas, es decir, una tarea no podrá comenzar hasta que no finalice la anterior. Permite exportar la tarea realizada a una imagen JPG, PNG, PDF y HTML.

Las características incluyen:

  • Crear diagramas de Gantt.
  • Crea y organiza tareas.
  • Crea tareas e hitos.
  • Dibuja restricciones de dependencia.
  • Crea líneas de base.
  • Generar gráficos PERT.
  • Crear y asignar recursos humanos.
  • Monitorear tareas asignadas.
  • Soporta WebDAV y almacenamiento en la nube.
  • Genere informes en PDF, como una imagen PNG / JPEG.
  • Exportar a CSV y Microsoft Project.
  • Importar desde formatos Microsoft Project e iCal.

GanttProject le permite programar sus proyectos con facilidad. Es una aplicación confiable y robusta que está repleta de herramientas y funciones útiles para permitirle administrar sus proyectos desde el primer momento.

Con GanttProject puede crear una estructura de desglose de trabajo, con la que puede organizar todas sus tareas bajo un mismo techo. A partir de ahí, puede crear un árbol de jerarquía para resumir y rastrear el progreso, los costos y las fechas y prácticamente cualquier otra cosa que deba monitorearse durante el proyecto. Las tareas se pueden ver cuando están en uso y se pueden contraer cuando no está trabajando en ellas.

GanttProject también es bastante flexible. Puede crear campos personalizados para sus proyectos, de esa manera no está limitado a las restricciones con las que la aplicación viene precargada. Las opciones de personalización no solo terminan con campos personalizados. Por ejemplo, puede relacionar una tarea con otra y establecer temporizadores para que las tareas comiencen cuando otras finalicen.

En general, GanttProject le proporciona todo tipo de herramientas de gestión de proyectos para utilizar. Desde establecer líneas de base, hasta recopilar datos de gráficos de informes y asignar tareas.

Nota: GanttProject está escrito en Java y requiere la instalación de Java Runtime versión 7 o superior.

Gestor de tiempos, enfocado principalmente en la generación de Diagramas de Gantt. Sustituye perfectamente a MS Project.
