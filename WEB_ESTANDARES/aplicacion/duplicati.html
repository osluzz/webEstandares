Duplicati es un programa de copias de seguridad de código abierto. A continuación se detallan sus principales características.

Muchos Backends:

Duplicati funciona con protocolos estándar como FTP, SSH, WebDAV, así como con servicios populares como Backblaze B2, Tardigrade, Microsoft OneDrive, Amazon S3, Google Drive, box.com, Mega, hubiC y muchos otros.

Características:

Realice copias de seguridad de archivos y carpetas con cifrado AES-256. Ahorre espacio con las copias de seguridad incrementales y la deduplicación de datos. Ejecute copias de seguridad en cualquier máquina a través de la interfaz basada en web o mediante la interfaz de línea de comandos. Duplicati incorpora un programador y un auto-actualizador.

Encriptación fuerte:

Duplicati utiliza una fuerte encriptación AES-256 para proteger su privacidad. También puede utilizar GPG para cifrar su copia de seguridad.

Construido para online:

Duplicati fue diseñado para copias de seguridad en línea desde cero. No sólo es eficiente con los datos, sino que también maneja muy bien los problemas de red. Por ejemplo, las copias de seguridad interrumpidas se pueden reanudar y Duplicati comprueba el contenido de las copias de seguridad con regularidad. De esta forma, las copias de seguridad rotas en sistemas de almacenamiento corruptos pueden detectarse antes de que sea demasiado tarde.

Interfaz de usuario basada en web:

Duplicati se configura mediante una interfaz web que se ejecuta en cualquier navegador (incluso móvil) y a la que se puede acceder -si se desea- desde cualquier lugar. Esto también permite ejecutar Duplicati en máquinas headless como un NAS (network attached storage).

Software libre:

Duplicati es software libre y de código abierto. Puede utilizar Duplicati gratuitamente incluso con fines comerciales. El código fuente está licenciado bajo LGPL. Duplicati funciona bajo Windows, Linux, MacOS. Requiere .NET 4.5 o Mono.
